## How to Capture Screenshots:

### Using Windows Snipping Tool:
1. Press `Win + Shift + S`
2. Select area to capture
3. Save as PNG

### Recommended Sizes:
- **Width:** 1920px
- **Height:** 1080px
- **Format:** PNG
- **Quality:** High

### Image Placement in PPT:
- **Position:** Center of slide
- **Size:** 80% of slide width
- **Border:** Subtle shadow effect
- **Caption:** Brief description below image
